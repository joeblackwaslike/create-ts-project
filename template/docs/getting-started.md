# Getting Started

## Installation

```bash
pnpm add __PROJECT_NAME__
```

## Usage

```typescript
import { greet } from '__PROJECT_NAME__';

console.log(greet('world'));
```

## Development

```bash
pnpm install    # install dependencies
pnpm test       # run tests
pnpm build      # build for production
pnpm docs:dev   # start docs server
```
