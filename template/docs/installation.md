# Installation

## Requirements

- Node.js `>= __NODE_VERSION__`
- A package manager: pnpm (recommended), npm, or bun

## Install

**pnpm (recommended)**

```bash
pnpm add __PROJECT_NAME__
```

**npm**

```bash
npm install __PROJECT_NAME__
```

**bun**

```bash
bun add __PROJECT_NAME__
```

## Verify

```typescript

// Replace with a real smoke-test import
```
